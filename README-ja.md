# Kubernetes + Multus CNI on OpenStack Demo

![CLI](./cli.gif)

## デモ環境の構成

* OpenStack
* セキュリティグループで必要なポートを開放(6443/tcp 30000-32767/tcp etc ...)
* Kubernetes 1.18 をOpenStackインスタンスで実行
* Multus CNI v3.6を利用
* FlannelもしくはCanalをデフォルトCNIとして利用
